# ⚡ Systematic Mode (정적 명령)

AI를 거치지 않고 시스템이 즉시 명령을 해석하여 실행합니다. 가장 빠르고 정확한 제어 방식입니다.

## 주요 특징
- **지연 시간 없음**: 자연어 처리 과정 없이 즉각 반응합니다.
- **정확한 매개변수**: 정해진 형식에 맞춰 데이터를 입력합니다.
- **신뢰성**: 복잡한 추론 없이 확정적인 결과를 보장합니다.

## 사용 방법
터미널이나 대화창에 `/`로 시작하는 명령어를 입력합니다.

```bash
/뉴스 브리핑
/할일 추가 "우유 사기"
/일정 목록
```

> [!TIP]
> 명령어 뒤에 인자를 생략하면 기본 동작(목록 보기 등)이 수행됩니다.
