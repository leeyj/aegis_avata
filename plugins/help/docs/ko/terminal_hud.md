# ⚡ 터미널 오버레이 HUD (Command Center)

AEGIS v2.0부터 도입된 터미널 방식 HUD를 통해 다양한 명령을 직관적으로 내릴 수 있습니다.

## ⌨️ 사용 방법
- **단축키**: 대시보드 어디서든 **`Shift + ~`** (Quake 방식)를 누르면 터미널 HUD가 나타납니다.
- **도움말**: `/help`를 입력하여 위젯 정보를 확인할 수 있습니다.

![터미널 HUD](/static/img/user_guide_terminal1.png)
*대시보드 어디서든 Shift + ~ 키로 호출*

## 🖥️ 통합 명령어 체계 (v3.4)
명령어 첫 글자에 따라 AI의 동작 모드가 결정됩니다.

1.  **🧠 하이브리드 추론 ( `/@` )**
    - 로컬 데이터와 실시간 웹 검색을 결합합니다.
    - 예: `/@오늘 내 일정과 어울리는 점심 메뉴 추천해줘`
2.  **🏠 로컬 전용 요약 ( `/` )**
    - 오직 시스템 내부 데이터만을 사용합니다. (보안 중시)
    - 예: `/뉴스`, `/일정 브리핑해줘`
3.  **🌐 확정적 웹 검색 ( `/#` )**
    - 로컬 데이터를 배제하고 순수 웹 검색만 수행합니다.
    - 예: `/#현재 엔비디아 주가 알려줘`
4.  **🤖 자율 대화 (접두사 없음)**
    - AI가 컨텍스트 활용 여부를 자율적으로 판단합니다.

## 💡 부가 옵션
- **음소거 (`--m` / `--mute`)**: 응답 끝에 붙이면 아바타가 말하지 않고 텍스트로만 출력합니다.
