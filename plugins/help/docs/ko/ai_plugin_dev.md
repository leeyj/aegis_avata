# 🤖 AI로 플러그인 개발하기

AEGIS는 **AI 에이전트가 플러그인을 직접 개발할 수 있도록** 설계되었습니다.

## 🚀 빠른 개발 루틴
1.  **골격 자동 생성**: 약 20개 규칙을 자동 준수합니다.
    ```bash
    python create_plugin.py --id my-widget --name "나의 위젯"
    ```
2.  **AI에게 맡기기**: `AI_AGENT_PROMPT.md`를 ChatGPT/Claude에 복사-붙여넣기 하세요.
    > "비트코인 실시간 가격 위젯 만들어줘" → 즉시 개발 가능

## ✨ 왜 AEGIS인가요?
- 📋 **구조화된 문서체계**: 모든 규칙·API·스키마가 단일 문서에 정의 (Single Source of Truth)
- 🤖 **AI 전용 프롬프트**: 전용 프롬프트를 통해 어떤 AI든 즉시 개발 가능
- 🛠️ **보일러플레이트 생성기**: 폴더 구조부터 Context API 연동까지 자동 적용
- 🔌 **플러그 앤 플레이**: 코어 수정 없이 폴더 추가/삭제만으로 확장 가능
