# 🧠 Hybrid Mode (지능형 결합)

플러그인이 가진 **실시간 데이터**와 **AI의 추론 능력**을 결합한 AEGIS의 핵심 모드입니다.

## 주요 특징
- **데이터 기반 답변**: AI가 단순히 지식으로 답하는 것이 아니라, 현재 등록된 할일이나 일정, 뉴스 데이터를 읽고 답변합니다.
- **유연한 처리**: "내일 오후 3시에 회의 잡아줘"와 같은 자연어를 읽고 적절한 플러그인 액션을 제안하거나 실행합니다.
- **Zero-Maintenance**: 각 플러그인별 `ai_prompt.md`를 통해 개별적인 인격을 가집니다.

## 사용 방법
`/@` 뒤에 플러그인 이름(또는 별명)을 붙여 질문합니다.

```bash
/@뉴스 오늘 주요 뉴스 중에 IT 관련만 요약해봐
/@할일 이번 주말까지 끝내야 하는 일들이 뭐가 있지?
/@일정 내일 오전 스케줄 알려줘
```
