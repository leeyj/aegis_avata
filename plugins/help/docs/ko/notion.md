# 🧠 노션 지식 허브 (Notion Intelligence Hub)

AEGIS는 파편화된 개인의 지식을 체계적으로 관리하고 AI가 학습할 수 있도록 돕는 **노션(Notion) 통합 솔루션**을 제공합니다.

## 🖼️ 관리 인터페이스

![Notion Cleanup 1](/static/img/notion_cleanup1.png)
*지능형 정리 규칙(Rules) 검토 및 미분류 항목 감지*

![Notion Cleanup 2](/static/img/notion_cleanup2.png)
*정리 결과에 대한 AI 지능형 요약 및 음성 브리핑*

## 🛠️ 주요 기능
1.  **퀵 캡처 (Quick Capture)**: 터미널 명령(`/n`, `/todo`)을 통해 생각난 즉시 메모를 기록합니다.
2.  **멀티 워크스페이스 💎**: 여러 개의 노션 워크스페이스를 별칭(`@업무`, `@개인`)으로 관리합니다.
3.  **지능형 정리 규칙 💎**: 키워드를 분석하여 속성을 자동으로 변경하고 데이터를 분류합니다.
4.  **AI 지식 브리핑**: 노션에 기록된 데이터를 AI가 분석하여 음성으로 요약해 줍니다.

> [!TIP]
> `/n` 명령어를 사용하여 터미널에서 즉시 노션으로 아이디어를 전송해 보세요.
