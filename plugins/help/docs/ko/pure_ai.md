# 🌐 Pure AI Mode (순수 지능)

플러그인 제약 없이 AI의 방대한 지식과 실시간 인터넷 검색을 활용하는 모드입니다.

## 주요 특징
- **무제한 지식**: 과학, 역사, 코딩 등 모든 분야에 대해 질문할 수 있습니다.
- **실시간 검색**: 시스템 내부에 정보가 없는 경우 구글 검색을 통해 최신 정보를 분석합니다.
- **감성 대화**: AEGIS 보이스 아바타와의 자유로운 감성 대화가 가능합니다.

## 사용 방법
명령어 앞에 `#`를 붙이거나, 태그 없이 질문하면 AI가 응답합니다.

```bash
# 양자 역학의 기본 원리에 대해 설명해줘
서울에서 가장 맛있는 평양냉면집 3곳 추천해줘 (검색 수행)
오늘 기분은 어때?
```

> [!IMPORTANT]
> 인텔리전스 검색이 활성화된 경우 분석에 시간이 다소 소요될 수 있습니다.
