# 🎨 Live2D 스튜디오 (Live2D Studio) 💎

AEGIS는 사용자가 직접 자신만의 Live2D 아바타를 최적화하고 관리할 수 있는 **전문가용 스튜디오 엔진**을 내장하고 있습니다.

## 🖼️ 주요 인터페이스

![Studio UI 1](/static/img/studio1.png)
*지능형 모델 분석 및 실시간 렌더링 화면*

![Studio UI 2](/static/img/studio2.png)
*에일리어스 매니저 및 지능형 키워드 매핑*

![Studio UI 3](/static/img/studio3.png)
*리액션 시뮬레이터 및 자동 최적화(Auto-Heal) 리포트*

## 🛠️ 주요 기능
1.  **자동 최적화 (Auto-Heal) 💎**: 구형 모델의 문법 오류나 경로 설정을 최신 규격으로 자동 수리합니다.
2.  **지능형 에일리어스 매니저**: 난해한 파일명을 `joy`, `shock`, `idle` 등 표준 별명에 매핑합니다.
3.  **리액션 시뮬레이터**: 주가 급등락, 날씨 변화 상황을 가상으로 발생시켜 반응을 검증합니다.
4.  **운영 배포 (Apply) 💎**: 설정을 완료한 모델을 즉시 대시보드 아바타로 교체합니다.

> [!NOTE]
> 일반 사용자는 시뮬레이션 및 분석 기능까지 체험 가능하며, 저장 및 배포 권한은 후원자(Sponsor)에게 제공됩니다.
