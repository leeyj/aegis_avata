# 💎 후원자 혜택 (Sponsorship Benefits)

AEGIS 프로젝트의 성장을 지원해 주시는 분들께는 전문가용 프리미엄 기능을 제공합니다.

## 💰 혜택 비교

| 분류 | 기능 | 일반 사용자 | 후원자 (Sponsor) |
|:---:|---|:---:|:---:|
| **스튜디오** | **Live2D 모델 에디터 및 리액션 시뮬레이터** | 테스트 모드 | **O (배포/저장 권한)** |
| **지식 허브** | **노션(Notion) 지능형 연동** | 단일 워크스페이스 | **멀티 워크스페이스 & 자동 정리** |
| **자동화** | **지능형 알리아스 자동 생성 도구** | X | **O (Sponsor Key)** |
| **댄스 모듈** | **YouTube Music 비트 싱크 댄스 로직** | 기본 | **고도화 예정** |

## 🚀 후원 티어
- **Standard Sponsor ($10/mo)**: 스튜디오 모든 저장/배포 기능 및 알리아스 자동 생성 해금.
- **Premium Advocate ($25/mo)**: 최우선 기능 요청 및 1:1 기술 지원.

> [!IMPORTANT]
> 후원은 [**GitHub Sponsors**](https://github.com/sponsors/leeyj)를 통해 진행하실 수 있으며, 전용 **Sponsor Key**를 발급해 드립니다.
