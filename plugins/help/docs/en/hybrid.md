# Hybrid Mode (🧠)

Hybrid mode combines real-time data from plugins with AI's strategic reasoning capabilities.

### Why Hybrid?
Pure AI often lacks real-time information, while Systematic mode requires strict syntax. Hybrid mode bridges this gap.

### Execution Flow
1. **Natural Input**: User types `/@ [query]`.
2. **AI Analysis**: AI identifies the intent and necessary plugin.
3. **Data Tooling**: AI fetches real-time data (e.g., current schedule, weather).
4. **Strategic Response**: AI provides an optimized answer or performs an action.

### Example
`/@todo What are my urgent tasks for today?`
-> AI analyzes your TODO list and highlights high-priority items.
