# Systematic Mode (⚡)

Systematic mode is the core command execution system that operates instantly without AI intervention.

### Key Features
- **Deterministic**: Executes exact logic for matching commands.
- **Zero Latency**: No waiting for AI generation.
- **Reliable**: Works even without an internet connection (for local plugins).

### How to Use
Type a command starting with a slash (`/`) in the terminal or HUD.
- Format: `/[plugin] [action] [params]`
- Example: `/alarm set 08:00 | Wake up`

> [!TIP]
> Use the pipe (`|`) symbol to separate multiple parameters.
