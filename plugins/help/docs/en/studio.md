## 🛠️ Key Tools
1.  **Auto-Heal 💎**: Automatically fixes syntax errors and path configurations in legacy models.
2.  **Alias Manager**: Maps complex filenames to standard aliases like `joy`, `shock`, or `idle`.
3.  **Reaction Simulator**: Virtually triggers scenarios (e.g., stock surges) to verify avatar responses.
4.  **Production Deployment (Apply) 💎**: Instantly swaps the dashboard avatar with your configured model.

> [!NOTE]
> Free users can experience analysis and simulation; saving and deployment permissions are for sponsors.
