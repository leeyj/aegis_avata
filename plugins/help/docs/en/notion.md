## 🛠️ Key Features
1.  **Quick Capture**: Record notes instantly to Notion using terminal commands like `/n` or `/todo`.
2.  **Multi-Workspace 💎**: Manage multiple Notion workspaces seamlessly using aliases (e.g., `@work`, `@personal`).
3.  **Intelligent Cleanup Rules 💎**: Analyze keywords to automatically update properties and classify data.
4.  **AI Knowledge Briefing**: AI analyzes your Notion records and provides a voice summary of your progress.

> [!TIP]
> Use the `/n` command to send ideas directly from the terminal to your Notion database.
