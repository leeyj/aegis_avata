# Pure AI Mode (🌐)

Pure AI mode focuses on creative generation, general knowledge, and complex problem-solving.

### Use Cases
- **Creative Writing**: Emails, reports, or code snippets.
- **Knowledge Base**: Questions about history, science, or general information.
- **Code Assistance**: Debugging or explaining complex logic.

### Interaction
Simply type your message without any prefix (unless configured otherwise).

> [!IMPORTANT]
> Pure AI does not have direct access to your local files or real-time plugin data unless you switch to **Hybrid** mode using the `/@` prefix.
