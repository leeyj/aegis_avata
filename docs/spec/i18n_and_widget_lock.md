# AEGIS Intelligence - 다국어 지원 및 위젯 잠금 가이드

본 문서는 **AEGIS Dashboard**의 주요 업그레이드 기능인 **다국어 지원(Internationalization)**과 **위젯 잠금(Widget Lock)** 기능의 사용법 및 구조를 설명합니다.

---

## 🌎 1. 다국어 지원 (i18n)

AEGIS는 현재 한국어(KO)와 영어(EN)를 공식 지원하며, 시스템의 모든 텍스트는 언어팩 파일에 의해 제어됩니다.

### 1.1 언어 전환 방법
- 사이드바(왼쪽 하단)의 **[KOREAN]** 또는 **[ENGLISH]** 버튼을 클릭합니다.
- 클릭 시 브라우저의 `localStorage`와 서버의 `config/system.json`에 설정이 즉시 저장되며 페이지가 새로고침됩니다.

### 1.2 언어팩 구조 (`config/i18n/`)
모든 번역 데이터는 JSON 형식으로 관리됩니다.
- `ko.json`: 한국어 번역 리소스
- `en.json`: 영어 번역 리소스
- **주요 섹션**:
  - `sidebar`: 메뉴 라벨
  - `panels`: 대시보드 위젯 타이틀
  - `directions`: 주가/지수 변동 방향성 (상승, 하락 등)
  - `studio`: Live2D 스튜디오 관련 UI 및 메시지
  - `terminal`: 명령 입력창 가이드

### 1.3 기술적 구현 (개발자 가이드)
- **HTML**: `data-i18n="키.경로"` 속성을 부여하면 페이지 로드 시 `I18nManager`가 자동으로 번역을 주입합니다.
  - 예시: `<h3 data-i18n="panels.p-clock">System Clock</h3>`
- **JavaScript**: 스크립트 내에서 동적으로 텍스트가 필요한 경우 전역 함수 `_t()`를 사용합니다.
  - 예시: `alert(_t('studio.messages.success'));`
- **Backend**: 시스템 언어에 따라 API 응답(예: 반응 규칙 설정)을 다르게 전달하기 위해 `_en.json` 접미사를 가진 설정 파일을 선호적으로 로드합니다.

## 🔒 2. 위젯 잠금 (Widget Lock)

대시보드의 위젯(Glass Panel)들은 자유롭게 드래그하거나 크기를 조절할 수 있습니다. 위젯 잠금 기능은 레이아웃이 확정된 후 실수로 위치가 바뀌는 것을 방지합니다.

### 2.1 사용 방법
- 사이드바 메뉴 상단의 **[LOCK WIDGETS]** (또는 **[위젯 위치 고정]**) 버튼을 클릭합니다.
- **잠금 활성화 시**:
  - 위젯을 마우스로 드래그할 수 없습니다.
  - 우측 하단의 리사이즈 핸들이 작동하지 않습니다.
  - 마우스 커서가 기본형으로 고정됩니다.
- **잠금 해제 시**: 다시 버튼을 누르면 자유로운 레이아웃 수정이 가능합니다.

### 2.2 영구 저장
- 위젯의 위치와 크기, 그리고 **잠금 상태**는 서버의 클라우드 설정에 저장되어 다음 접속 시에도 그대로 유지됩니다.
