# AEGIS 노션 지식 허브 (Notion Intelligence Hub) 가이드

AEGIS는 파편화된 개인의 지식을 체계적으로 관리하고 AI가 학습할 수 있도록 돕는 **노션(Notion) 통합 솔루션**을 제공합니다. 

터미널 명령 한 번으로 생각을 기록하고, AI가 알아서 지식을 정리하여 브리핑해주는 스마트한 환경을 설정할 수 있습니다.

---

## 1. 노션 API 연동 및 기본 설정 방법

AEGIS의 노션 기능을 사용하기 위해서는 먼저 노션의 관리자 토큰과 연결할 데이터베이스의 ID(주소)가 필요합니다.

### 1.1 API 토큰 발급 (Internal Integration Secret)
1. [Notion Developers 연동 관리 페이지](https://www.notion.so/my-integrations)에 접속합니다.
2. 좌측 메뉴에서 **[새 API 통합(New Integration)]** 버튼을 클릭합니다.
3. 적절한 이름(예: `AEGIS`)을 입력하고 저장합니다.
4. 생성이 완료되면 화면에 나타나는 **"프라이빗 API 통합 토큰(Internal Integration Secret)"**을 복사합니다.
5. 시스템 설정 파일(`config/secrets.json` 등)에서 `NOTION_API_KEY` 환경 변수에 복사해둔 토큰을 입력합니다.

### 1.2 데이터베이스(Database) ID 연결
1. 권한 부여: 본인의 노션 앱(또는 웹)으로 접속하여, 연동하고자 하는 데이터베이스 페이지를 엽니다.
2. 우측 상단의 `...` 메뉴를 클릭하고 **[연결(Connect)]**을 눌러, 앞서 생성한 통합(예: `AEGIS`) 이름을 검색하여 추가(권한 부여)합니다.
3. 데이터베이스 ID 복사: 웹 브라우저 주소창(또는 '링크 복사')의 URL을 보면 일련번호(UUID) 문자열이 포함되어 있습니다. 이것이 `Database ID`입니다.
4. `plugins/notion/config.json` 파일을 찾아 `"workspaces"` 항목에서 `"id"` 옆에 복사한 Database ID를 붙여 넣습니다.

## 2. 주요 제공 기능 및 활용

설정이 올바르게 추가되었다면 대시보드와 터미널을 통해 막강한 지식 허브 기능을 활용할 수 있습니다.

### 2.1 퀵 캡처 (Quick Capture)
웹 브라우저에서 작업 중이거나 대시보드를 보는 도중 생각난 메모나 할 일을 터미널을 통해 즉시 노션 데이터베이스에 기록할 수 있습니다.
- **메모 기록 예시**: `단축키(Shift + ~) 호출 후 -> /n 이번 주말까지 문서 초안 완료하기`
- **할 일 기록 예시**: `/todo 오늘 마트에서 우유 사기`

### 2.2 멀티 워크스페이스 연동 (💎 후원자 전용)
업무용, 개인용, 스크랩용 등 여러 개의 노션 데이터베이스를 분리해 사용하는 경우 워크스페이스를 별칭 형태로 등록해 관리합니다.
- `notion.json` (또는 해당 플러그인의 `config.json`)의 `workspaces` 목록에 각각의 공간을 추가하고, `@업무`, `@개인` 등으로 **알리아스(alias)**를 부여합니다.
- 터미널에서 대상을 전환하고 싶을 때 명령 한 번으로 변경할 수 있습니다. (예: `switch`)

### 2.3 지능형 정리 규칙 (Cleanup Rules) (💎 후원자 전용)
사용자가 직접 작성한 메모나 기사 스크랩들을 AI가 미리 정해진 규칙에 따라 속성을 변경하고 저장 공간을 재분류합니다.
- 예: 제목의 키워드(`[폐기]`, `회의록`)를 분석해 폴더 이동/태그 설정을 실행합니다.
- 설정 방법: `notion_rules.json`에서 발동 조건과 수행할 분류 Action을 선언합니다.

### 2.4 AI 지식 브리핑
기록된 최근 메모와 지식 베이스를 AI가 검색 및 체계화하여 사용자에게 음성(TTS)으로 브리핑해줍니다. 진행 상황이나 우선순위를 식별하는데 유용합니다.
- 커스텀 RAG(검색 증강 생성) 방식으로 노션 데이터를 프롬프트에 동적으로 연결합니다. (v2.2.0 변경 사항 참조)
