## 💡 명령어 시스템 가이드 (Systematic & Hybrid)

### 1. Systematic Mode (⚡)
AI를 거치지 않고 직접 명령을 해석하여 즉시 실행합니다.
- **포맷**: `/[플러그인명] [액션] [파라미터]`
- **파라미터 구분**: 인자가 여러 개인 경우 파이프(`|`) 기호를 사용합니다.
  - 예: `/알람 추가 08:00 | 상쾌한 아침 기상`
- **단축 실행**: 사이드바 등을 통해 해당 플러그인에 '포커스'된 상태라면 플러그인명을 생략하고 액션만 입력하여 실행할 수 있습니다.

### 2. Hybrid Mode (🧠)
실시간 데이터와 AI의 추론 능력을 결합한 대화형 실행 모드입니다.
- **포맷**: `/@ [플러그인명] [자연어 질의]`
- **특징**: 파라미터 규격을 정확히 맞추지 않아도 AI가 문맥을 파악하여 액션을 제안하거나 수행합니다.
  - 예: `/@할일 이번 주말까지 끝내야 하는 업무 정리해줘` -> `list` 액션 수행 시 분석 결과 포함
  - 예: `/@알람 내일 오전 8시에 운동 예약` -> `set` 액션으로 자동 정규화
