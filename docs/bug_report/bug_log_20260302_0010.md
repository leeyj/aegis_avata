# AEGIS Bug Report & Resolution - [2026-03-02 00:10]

이 리포트는 Plugin-X 구조 고도화 및 시스템 안정성 확보 과정에서 발생한 버그와 조치 내용을 기록합니다.

---

### 1. [Critical] 배경화면 설정 저장 로직 오염 (Config Pollution)
- **상태**: 배경화면 변경 후 "SAVE SYNC" 시 새로고침하면 비디오 재생 실패 또는 검은 화면 발생.
- **원인**: `save_json_config` 함수가 기존 설정을 **병합(Merge)**하는 방식이 문제. 모드가 전환되어도 이전 모드의 특수 플래그(예: `is_video: true`)가 삭제되지 않고 남아있어, 이미지 배경을 비디오로 판단하는 논리 오류 발생.
- **조치**: 
    - `plugins/wallpaper/router.py`의 `save_json_config` 호출 시 `merge=False` 옵션 적용 (현재 상태만 정밀 기록).
    - `static/js/wallpaper/index.js`에서 모드 변경 시 관련 플래그 초기화 로직 보강.

### 2. [Major] 모듈 로딩 실패 (MIME Type Mismatch on Windows)
- **상태**: 타이틀 위젯 등 일부 플러그인의 JS 모듈이 `Failed to load module script` 오류와 함께 작동하지 않음.
- **원인**: Windows 환경에서 Flask의 `send_from_directory`가 `.js` 파일을 `text/plain`으로 전송하여 브라우저의 보안 정책(Strict MIME Type Checking)에 걸림.
- **조치**: 
    - `routes/plugins.py`에서 자산 서빙 시 `.js`는 `application/javascript`, `.css`는 `text/css`로 MIME 타입을 강제 명시하도록 수정.

### 3. [UX] 드래그 엔진과 클릭 이벤트 충돌
- **상태**: 타이틀의 브리핑 아이콘을 클릭해도 패널 이동(Drag)이 먼저 발동하여 클릭 이벤트가 무시됨.
- **원인**: `ui_drag_manager.js`의 인터랙티브 요소 감지 목록에 새로운 플러그인 아이콘(`.brief-speaker`)이 누락되어 드래그 엔진이 이벤트를 독점함.
- **조치**: 
    - `ui_drag_manager.js`에 `.brief-speaker` 및 `.no-drag` 예외 처리 추가.
    - 타이틀 HTML의 아이콘 요소에 `no-drag` 클래스 명시.

### 4. [System] 사이드바 토글 상태 불일치
- **상태**: 대시보드에서 위젯을 꺼놓아도 사이드바를 열면 모든 스위치가 `ON`으로 보임.
- **원인**: 사이드바 생성 로직이 `window.panelVisibility` 상태를 참조하지 않고 기본값(`checked`)으로만 생성됨.
- **조치**: 
    - `ui_sidebar_manager.js`에서 `panelVisibility` 캐시를 확인하여 체크박스의 초기 상태를 동적으로 설정하도록 수정.

### 5. [Architecture] 음성 출력(TTS) 서비스 유실
- **상태**: 모듈화된 플러그인 환경에서 `speakTTS` 함수를 찾지 못하거나 호출이 씹히는 현상 발생.
- **원인**: 비동기 로딩 순서에 따라 `speakTTS`가 전역(`window`)에 등록되기 전 플러그인이 호출을 시도함.
- **조치**: 
    - `static/js/widgets/tts.js` 끝단에서 `window.speakTTS` 명시적 노출.
    - 전 브리핑 과정(로딩-분석-수신-출력)에 상세 디버그 로그(`[ACTION]`, `[NETWORK]`, `[DEBUG]`) 시스템 구축.

### 6. [UIUX] 브리핑 피드백 디자인 어색함
- **상태**: 브리핑 실행 시 아이콘 텍스트가 깨지거나 "ANALYZING..." 글자가 아이콘 위에 겹침.
- **조치**: 
    - 타이틀 위젯에 **브리핑 허브(Briefing Hub)** 레이아웃 도입.
    - 스피커 아이콘은 유지하되, 왼쪽 전용 영역에서 텍스트 피드백이 나타나도록 분리.
    - 전술적 펄스 애니메이션 보강.
