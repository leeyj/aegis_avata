# AEGIS Plugin-X: 5-Minute Quick Start

> This document is the **shortest path for developers who want to build a plugin right now**.  
> For full rules and API details, see [`PLUGIN_SPEC_en.md`](PLUGIN_SPEC_en.md).

---

## Step 1: Generate Scaffolding (30 seconds)

Run this in the project root:

```bash
python create_plugin.py --id my-widget --name "My Widget" --lang en
```

> 💡 `--lang en` generates English comments and UI text. Without it, the default language is Korean.

This single command auto-generates:

```
plugins/my-widget/
├── __init__.py            # Package declaration (empty, do not delete)
├── manifest.json          # Metadata, permissions, exports
├── config.json            # Plugin settings storage
├── router.py              # Flask Blueprint (backend API)
├── my_widget_service.py   # Core business logic
└── assets/
    ├── widget.html        # UI skeleton (Shadow DOM)
    ├── widget.js          # Frontend logic (init/destroy lifecycle)
    └── widget.css         # Styles (Shadow DOM isolated)
```

### Common Options

```bash
# Plugin with permissions
python create_plugin.py --id stock-alert --name "Stock Alert" --lang en \
    --permissions api.ai_gateway \
    --csp-domains "https://api.example.com"

# Backend-only service (no UI)
python create_plugin.py --id bg-worker --name "Background Worker" --lang en --hidden

# Frontend-only widget (no backend)
python create_plugin.py --id simple-clock --name "Clock" --lang en --no-backend
```

## Step 2: Implement Business Logic (3 minutes)

Edit only the `TODO`-marked sections in the generated files.

### Backend: `my_widget_service.py`

```python
class MyWidgetService:
    @staticmethod
    def get_data(config: dict = None) -> dict:
        # TODO: Implement your actual logic here
        # e.g., external API calls, DB queries, file parsing
        return {
            "value": 42,
            "message": "Data loaded successfully"
        }

    @staticmethod
    def get_status() -> dict:
        # TODO: Status summary for AI briefing
        return {"summary": "My Widget is operating normally."}
```

### Frontend: `assets/widget.js`

Update the `refresh()` method to bind data to your UI:

```javascript
async refresh() {
    const res = await fetch('/api/plugins/my-widget/data');
    const data = await res.json();
    const el = this.shadow.querySelector('#status-text');
    if (el) el.textContent = data.message;
}
```

---

## Step 3: Verify (30 seconds)

1. Restart the Flask server.
2. Refresh the dashboard in your browser.
3. "My Widget" automatically appears in the sidebar.
4. Type `/my-widget status` in the terminal — the command works out of the box.
