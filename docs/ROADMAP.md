# AEGIS 시스템 - 향후 개발 로드맵 (Roadmap)

본 문서는 AEGIS 대시보드의 기능 고도화 내역과 향후 개발 계획을 통합하여 관리합니다.

---

## ✅ 완료된 주요 마일스톤 (Milestones)

### 0. 시스템 기반 강화 [완료]
- **모듈화 리팩토링 (Modular Engine v4.8) [완료]**: 
    - 백엔드 라우트 분리 및 명시적 임포트 전환.
    - 프론트엔드 HTML/CSS 컴포넌트화 완료.
    - 설정 로딩 시스템 통합 및 안전성 강화.
- **보안 및 민감 정보 분리 [완료]**:
    - `secrets.json` 도입으로 API 키(Gemini), 접속 계정, 보안 키 격리 관리.

### 1. 액션 및 프롬프트 엔진 고도화 [완료]
- **리액션 엔진 통합 (ReactionEngine) [완료]:**
    - `reactions.json`을 통한 조건부 액션(MOTION, EMOTION, TTS) 통합 관리.
- **AI 프롬프트 외부화 [완료]:**
    - `prompts.json` 신설 (Briefing, Command, Proactive 설정 분리).

### 2. 의미론적 알리아스(Semantic Alias) 시스템 구축 [완료]
- **범용 아바타 제어 인프라 [완료]**:
    - 모델별 파일명 규칙에 상관없이 `idle`, `joy` 등 공통 키워드로 제어하는 시스템 구축.
    - 백엔드(`utils.py`) 및 프론트엔드(`AnimationManager`) 연동 완료.
- **지능형 추론 및 검증 도구 [완료]**:
    - `check_assets.py` (자동 알리아스 생성) 및 `check_missing_alias.py` 구축.
    - 다국어(한/중/일) 키워드 대응 매칭률 100% 달성.

### 3. 멀티미디어 및 시각화 [완료]
- **오디오 비주얼라이저 [완료]**: TTS 출력에 동기화되는 실시간 주파수 시각화 엔진 구축.
- **음성 브리핑 연동 [완료]**: Edge-TTS 활용 실시간 음성 생성 및 오디오 캐싱 시스템 구축.

---

## 🚀 향후 개발 계획 (Ongoing & Future)

### 1. AI 기반 뉴스 브리핑 고도화 [진행 중]
- **Google Gemini API 통합 [완료]**: Gemini 2.0 Flash 기반 요약 엔진 구축.
- **뉴스 종합 요약 [대기]**: RSS 데이터를 Gemini 요약 프롬프트에 동기화하는 로직 완성.

### 2. 사용자 편의성 및 인터페이스 확장
- **실시간 위젯 에디터 [대기]**: 브라우저 UI 상에서 위젯 설정(크기, 폰트 등)을 변경하는 설정 대시보드.
- **모바일 최적화 뷰포트 [대기]**: 다양한 모바일 기기 대응 레이아웃 개선.

### 3. 라이브 상호작용 심화
- **마이크 입력 반응**: 사용자 음성 감지 시 아바타의 경청 모션 자동화.
- **멀티 아바타 모드**: 시간대나 상황에 따른 아바타 자동 전환 시스템.
- **이벤트 버스 구조 심화**: 모든 위젯과 아바타 동작을 통합 제어하는 중앙 이벤트 관리 강화.
