## 🚀 향후 개발 계획 (Ongoing & Future)

### 1. v1.2: 로컬 지식 및 기반 최적화 [진행 중]
- ~~**Live2D 스튜디오 최적화 엔진**~~: ~~[완료] 구형 자산 복구 및 지능형 에일리어스 연동 도구 통합.~~
- **개인지식 학습 엔진**: 개인이 기록한 파편화된 지식(PKM)을 AEGIS가 학습하고 브리핑하는 "나의 기억을 공유하는 비서"로 고도화합니다.
- **옵시디언(Obsidian) 로컬 연동**:
    - `Obsidian Local REST API` 및 직접 파일 시스템 접근을 통한 노트 검색/읽기.
    - AI 질의 시 관련 노트를 참조하여 답변하는 RAG(검색 증강 생성) 기반 지식 주입.

### 2. v1.3: 글로벌 확장 및 서비스 차별화 (Global & Service Optimization)
- **영어 및 다국어 지원 (Global Support) [1순위]**:
    - **UI 로컬라이징**: 대시보드 및 스튜디오 인터페이스의 영어 버전 제공.
    - **다국어 브리핑**: AI 브리핑 및 알림 TTS의 영어 출력 기능 (Edge-TTS 영어 음성 매핑).
    - **언어 선택 토글**: 설정 창에서 한국어/영어 즉시 전환 기능.
- **노션 지능형 부분 유료화(Freemium) 정식 적용**:
    - **멀티 워크스페이스 제어**: 일반 사용자는 단일 계정 연동, 스폰서는 무제한 멀티 워크스페이스/데이터베이스 연동 기능 해금.
    - **규칙 기반 자동 정리 엔진**: `notion_rules.json`을 활용한 제목 분석 및 속성 자동 업데이트 기능을 스폰서 전용 프리미엄 기능으로 전환.
    - **RAG 기반 지식 브리핑 고도화**: 스폰서 사용자의 경우 전체 노션 데이터베이스를 고수준으로 분석하여 개인 맞춤형 리포트를 생성하는 전용 LLM 워크플로우 제공.
- **위젯 레이아웃 잠금 (Layout Lock)**: 브라우저별로 설정된 위젯 위치가 실수로 변경되지 않도록 고정하는 기능.
- **프리미엄 테마 시스템 💎**: 다양한 UI 스타일(Neon, Cyber, Minimal 등)을 선택할 수 있는 테마 엔진 (스폰서 전용).
- **반응형 레이아웃 최적화 (Mobile/Tablet)**: 모바일과 태블릿의 기기 특성에 맞춘 차별화된 그리드 시스템 및 터치 인터랙션 로직 적용.
- **실시간 위젯 에디터**: 브라우저 UI 상에서 위젯 설정(크기, 폰트 등)을 변경하는 설정 대시보드.


### 3. 라이브 상호작용 및 오토메이션
- **마이크 입력 반응**: 사용자 음성 감지 시 아바타의 경청 모션 및 시선 처리 자동화.
- **멀티 아바타 모드**: 시간대나 상황에 따라 아바타 및 테마 자동 전환 시스템.
- **중앙 이벤트 관리**: 모든 위젯과 아바타 동작을 통합 제어하는 이벤트 버스 구조 심화.

## 📊 기능별 구현 난이도 분석 (Development Difficulty)

| 고도화 항목 | 난이도 | 주요 기술적 도전 과제 |
| :--- | :---: | :--- |
| **영어 및 다국어 지원 (1순위)** | **중(Medium)** | 모든 UI 하드코딩 텍스트 추출 및 언어별 프롬프트/TTS 음성 매핑 관리. |
| **노션 지능형 부분 유료화 (Freemium)** | **중(Medium)** | 스폰서 키 검증 로직 및 프론트/백엔드 기능 잠금(Lock) 동기화 처리. |
| **반응형 레이아웃 최적화 (Mobile/Tablet)** | **중상(High)** | 기기별 상이한 뷰포트에 따른 위젯 재배치 및 터치 인터랙션(드래그) 예외 처리. |
| **프리미엄 테마 시스템 💎** | **중(Medium)** | CSS 변수(Variables) 기반의 동적 테마 교체 엔진 및 설정 유지 시스템 구축. |
| **위젯 레이아웃 잠금 (Layout Lock)** | **하(Easy)** | 로컬 스토리지 또는 설정 파일에 드래그 가능 여부(`draggable`) 플래그 연동. |
| **옵시디언 로컬 연동** | **중상(High)** | 로컬 파일 시스템 또는 REST API 직접 접근 및 RAG 기반 지식 주입 파이프라인 구축. |
| **실시간 위젯 에디터** | **중(Medium)** | 폰트/색상/크기 설정을 실시간으로 반영하고 JSON 설정에 영구 저장하는 UI 구현. |
| **마이크 입력 반응** | **상(Critical)** | 브라우저 Web Speech API 연동 및 음성 감지 시 애니메이션 트리거링 동기화. |
